# Hotel Booking Data Analysis

## Project Overview
This project analyzes hotel booking data using Excel and Power BI to identify booking trends, cancellations, customer behavior, and business performance.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Dataset
Hotel Booking Dataset

## Dashboard Features
- Total Bookings
- Cancellation Analysis
- Customer Type Analysis
- Country-wise Bookings
- ADR Analysis
- Monthly Booking Trends

## Key Insights
- City Hotels receive more bookings than Resort Hotels.
- Cancellation rates are high among some customer groups.
- Most bookings occur during peak travel seasons.
- Repeat guests have a lower cancellation rate.

## Recommendations
- Reduce cancellations with flexible booking offers.
- Reward repeat customers through loyalty programs.
- Increase marketing during off-season months.
- Improve pricing strategy during peak seasons.
